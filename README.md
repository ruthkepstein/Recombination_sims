All data in this repository relates to simulations conducted in maize & rice. Each mutant (ie ddm1, zmet2, fancm recq4, 10X, ddm1/zmet2) has its own .R file with the construction of genetic maps.
"Ideal1" is 10x increase in recombination and "Ideal2" is hypothetical ddm1/zmet2 mutant.

Code for simulations ran (both recurrent selection & introgression) are in intermating & introgression marked files. Plots for results can be found at bottoms of those .R files.
General plots like for genetic maps can be found in Plots.R.
If file does not have a species in its name, assume maize. All rice-related files has "rice" in the name. All simulations were done with Japonica, even though Indica genetic maps were created and gene density in Indica was analyzed.

To reproduce our data, set seed at 420. If you want original genetic maps or recombination data used for ddm1, zmet2, recq4, or fancm, email rke27@cornell.edu. Wildtype recombination data for maize is avaliable in NAM .csv file.
Gene density was found from .gff in maize and rice. Correlations between recombination rate and gene density can be found in gene space R files.

Simulations related to selection intensities can be found in selection intensities R file. This was only done in maize.

Targeted recombination simulations can be found in targeted sims and were only done in maize.

If you have questions or want data, email rke27@cornell.edu for maize and ns623@cornell.edu for rice data.
